## Monetra

Smart AI for bill tracking and expense monitoring.

#### Features

- Scans Gmail for bills and invoices  
- Extracts amount, due date, and category using AI  
- Stores structured data in database  
- Sends due-date reminders  
- Detects duplicate subscriptions  
- Flags unusual spending spikes  
- Generates weekly expense summaries  
- Helps avoid late fees and overspending
