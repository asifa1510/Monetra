## Monetra

Smart AI system for automated bill tracking and expense intelligence.

#### Features

- Automatically scans Gmail for bills, invoices, and subscriptions  
- Extracts amount, due date, category, and recurrence using AI  
- Stores structured financial records in database  
- Sends proactive due-date reminders  
- Detects duplicate and inactive subscriptions  
- Flags abnormal spending spikes  
- Generates weekly financial summaries  
- Provides basic financial risk insights  

#### Tech Stack

- n8n – Workflow orchestration  
- Google Gemini API – AI-based extraction and reasoning  
- Gmail API – Email monitoring  
- Google Sheets / PostgreSQL – Data storage  
- Telegram API – Real-time alerts  
- Cron Jobs – Scheduled reports
